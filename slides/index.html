Welcome to blendOS!


blendOS is a seamless blend of all your Linux distributions. Click on Get Started, to put it simply, get started.



Immutable.


Here's something you may not have known: blendOS is an immutable operating system.



Color scheme


Select a color scheme (light or dark).





Use apps from any distribution


Just open Console, where you'll find how to install apps from any distro in the main session.



Install Flatpaks


You can also install any Flatpak from Flathub. Just open the Flathub Store. (note: app removal isn't currently supported through the store, so you'll need to use the flatpak command for that)
